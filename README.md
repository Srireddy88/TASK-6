##  Steps to Host using GitHub Pages

**Created a GitHub Repository**
**name : TASK-6**
**Files Added**
   - `index.html`, `style.css`, `script.js`.

**GitHub Pages**
   - Go to your repository → **Settings** → **Pages**.
     - **Branch**: `main`
     - **Folder**: `/ (root)`
    - Click **Save**.
